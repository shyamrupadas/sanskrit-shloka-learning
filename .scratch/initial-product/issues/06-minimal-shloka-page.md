# Минимальная страница шлоки

Status: ready-for-human
Accepted: 2026-06-27

## Родитель

`.scratch/initial-product/PRD.md`

## Что сделать

Добавить MVP-страницу шлоки, доступную из карточек библиотеки. Страница должна оставаться намеренно минимальной: отображаемое название шлоки, текст шлоки кириллицей, возврат в библиотеку и общая мобильная оболочка.

## Критерии приемки

- [x] Пользователь может открыть страницу шлоки по коду шлоки из тела карточки библиотеки или стрелки перехода.
- [x] Страница показывает отображаемое название шлоки и канонический текст шлоки кириллицей.
- [x] Страница включает понятный возврат в библиотеку и сохраняет общую нижнюю навигацию.
- [x] MVP-страница не показывает личный статус, пословную транслитерацию, пословный перевод, полный перевод шлоки или действия по статусу.
- [x] Тесты покрывают открытие из библиотеки, прямой маршрут по коду, отображение кириллического текста, возврат в библиотеку, наличие мобильной оболочки и отсутствие вынесенных за рамки расширенных сведений/действий.
