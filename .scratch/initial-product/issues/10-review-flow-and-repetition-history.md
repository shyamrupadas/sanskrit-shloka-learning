# Поток повторения и история повторений

Status: ready-for-agent

## Родитель

`.scratch/initial-product/PRD.md`

## Что сделать

Сделать поток повторения из дашборда и из карточек `Повторяю` в библиотеке. Поток должен проводить пользователя через вспоминание, подсказки, раскрытие полного текста, выбор результата и создание записей истории повторений только для завершенных повторений; незавершенные попытки не должны иметь побочных эффектов.

## Критерии приемки

- [ ] Пользователь может начать повторение со шлоки к повторению сегодня на дашборде или через `Повторить` на карточке `Повторяю`.
- [ ] Первый экран просит пользователя произнести шлоку по памяти до раскрытия текста.
- [ ] Последовательность подсказок фиксирована: первая половина первой строки, затем вся первая строка, затем весь текст.
- [ ] Открытие полного текста после неудачного вспоминания записывает результат `не смог вспомнить`.
- [ ] Если пользователь вспомнил самостоятельно, приложение показывает полный текст и требует выбрать `Все правильно` или `Сделал ошибку` до завершения.
- [ ] `Все правильно` записывает `вспомнил без ошибки`; `Сделал ошибку` записывает `вспомнил с ошибкой`; завершение после подсказки записывает `вспомнил с подсказкой`.
- [ ] Завершенная запись истории повторений включает пользователя, шлоку, время завершения, день пользователя и результат.
- [ ] Выход из потока повторения до завершения не создает запись истории и не убирает шлоку из сегодняшних кандидатов.
- [ ] После завершения одной шлоки поток открывает следующую шлоку в текущем потоке повторения, если она есть; иначе возвращает к состоянию завершения на дашборде.
- [ ] Тесты покрывают все четыре результата, вход с дашборда, вход из библиотеки, незавершенный выход без побочных эффектов, переход к следующей шлоке текущего потока и завершение дашборда после последнего кандидата.
