# Статический раздел `Обучение санскриту`

Status: ready-for-agent

## Родитель

`.scratch/initial-product/PRD.md`

## Что сделать

Добавить раздел `Обучение санскриту` как статический список раскрывающихся статей-советов. В первой реализации можно использовать временный текст, написанный разработчиком; структура и навигация должны позволять позже заменить текст редакторски без добавления CMS.

## Критерии приемки

- [ ] Раздел доступен из нижней мобильной навигации с короткой подписью `Обучение`.
- [ ] Заголовок страницы использует каноническое название `Обучение санскриту`.
- [ ] Страница показывает раскрывающиеся секции советов, например про чтение шлок по строкам, заучивание через смысл слов и действия, если шлока не вспоминается.
- [ ] Временный текст статей написан на русском и не заявляет в UI или комментариях, что является финальным редакторским содержанием.
- [ ] В разделе нет CMS, отслеживания прогресса, структуры курсов, квизов или административного редактирования.
- [ ] Тесты покрывают переход в раздел, раскрытие/сворачивание советов, русские UI-строки через i18n и отсутствие курсов/квизов/прогресса.
