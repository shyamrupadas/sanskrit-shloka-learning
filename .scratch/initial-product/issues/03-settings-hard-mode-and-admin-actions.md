# Настройки, hard mode и админские действия

Status: ready-for-agent

## Родитель

`docs/prd/0001-initial-product.md`

## Что сделать

Добавить экран настроек учетной записи для hard mode, выхода и административных действий текущего этапа. Кнопки `Добавить источник` и `Добавить шлоку` размещаются в настройках, видны только администраторам и открывают защищенные административные формы из среза создания источника и шлоки.

## Критерии приемки

- [ ] Экран настроек доступен авторизованным пользователям из нижней мобильной навигации.
- [ ] Пользователь может переключить hard mode, и выбор сохраняется после перезагрузки и между сессиями.
- [ ] Hard mode хранится как предпочтение, но не меняет поведение повторения в MVP.
- [ ] Экран настроек включает выход из учетной записи.
- [ ] Экран настроек показывает `Добавить источник` и `Добавить шлоку` только пользователям с ролью администратора.
- [ ] Действия `Добавить источник` и `Добавить шлоку` открывают соответствующие защищенные формы; отдельный экран списка админки не появляется.
- [ ] Тесты покрывают чтение и сохранение hard mode, выход из настроек, видимость административных действий только для администратора, переход к защищенным формам и отсутствие влияния hard mode на повторение в MVP.
