# Настройки, hard mode и вход в админку

Status: ready-for-human
Accepted: 2026-06-27

## Родитель

`.scratch/initial-product/PRD.md`

## Что сделать

Добавить экран настроек учетной записи для hard mode, выхода и входа в административный раздел. Действие `Админка` размещается в настройках, видно только администраторам и открывает защищенную страницу админки со списком шлок.

## Критерии приемки

- [x] Экран настроек доступен авторизованным пользователям из нижней мобильной навигации.
- [x] Пользователь может переключить hard mode, и выбор сохраняется после перезагрузки и между сессиями.
- [x] Hard mode хранится как предпочтение, но не меняет поведение повторения в MVP.
- [x] Экран настроек включает выход из учетной записи.
- [x] Экран настроек показывает действие `Админка` только пользователям с ролью администратора.
- [x] Действие `Админка` открывает защищенную страницу админки со списком шлок.
- [x] Тесты покрывают чтение и сохранение hard mode, выход из настроек, видимость входа в админку только для администратора, переход к защищенной админке и отсутствие влияния hard mode на повторение в MVP.
