# Настройки, hard mode и вход в админку

Status: ready-for-agent

## Родитель

`docs/prd/0001-initial-product.md`

## Что сделать

Добавить экран настроек учетной записи для hard mode, выхода и входа в административный раздел. Действие `Админка` размещается в настройках, видно только администраторам и открывает защищенную страницу админки со списком шлок.

## Критерии приемки

- [ ] Экран настроек доступен авторизованным пользователям из нижней мобильной навигации.
- [ ] Пользователь может переключить hard mode, и выбор сохраняется после перезагрузки и между сессиями.
- [ ] Hard mode хранится как предпочтение, но не меняет поведение повторения в MVP.
- [ ] Экран настроек включает выход из учетной записи.
- [ ] Экран настроек показывает действие `Админка` только пользователям с ролью администратора.
- [ ] Действие `Админка` открывает защищенную страницу админки со списком шлок.
- [ ] Тесты покрывают чтение и сохранение hard mode, выход из настроек, видимость входа в админку только для администратора, переход к защищенной админке и отсутствие влияния hard mode на повторение в MVP.
