# 05 — Перенести Общую библиотеку шлок и страницу Шлоки

**What to build:** Перенести полный library-сценарий в одну фичу: загрузка Общей библиотеки шлок, табы, Поиск шлок, изменение Статуса шлоки пользователя и открытие минимальной страницы Шлоки.

Внутри фичи отделить API state и пользовательские действия от отображения через небольшой интерфейс. Чистую логику поиска, фильтрации и выбора действия оставить тестируемой без React, не превращая каждый helper в отдельный модуль.
