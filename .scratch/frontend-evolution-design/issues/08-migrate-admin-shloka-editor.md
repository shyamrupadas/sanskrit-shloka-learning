# 08 — Перенести редактор Шлоки

**What to build:** Добавить в `admin` полный подмодуль `shloka-editor`, который обслуживает создание и редактирование Шлоки через общую внутреннюю модель формы. Модель должна скрывать выбор места в Структуре источника шлоки, нормализацию Пад шлоки, validation, immutable reference fields и преобразование request.

Подмодуль переиспользует внутренние admin UI-элементы, сформированные предыдущим срезом, но не расширяет public API фичи внутренними полями или helpers.

**Blocked by:** None — can start immediately

**Status:** awaiting-human-review
Accepted: 2026-07-05
