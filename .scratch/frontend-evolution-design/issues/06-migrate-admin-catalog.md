# 06 — Перенести каталог Админки

**What to build:** Создать фичу `admin` первым административным tracer bullet: защищенный каталог загружает Источники шлоки и вложенные Шлоки, отображает прежние подписи и предоставляет переходы к созданию и редактированию.

Каталог становится первым внутренним подмодулем `admin`. Его UI, query state и форматирование скрыты за route-level public API фичи. Редакторы Источника шлоки и Шлоки пока могут оставаться legacy routes, но не должны импортировать внутренности нового каталога.

**Blocked by:** None — can start immediately
